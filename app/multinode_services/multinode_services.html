Multinode Services

Available Services

Before submitting a cluster, a service from the ones available below has to be selected. The cluster can be customized with a number of nodes, CPU's per node and memory per node. You can create a cluster that fits your needs or the service to be deployed.

Slurm

Create a cluster with Slurm deployed and configured, ready to run jobs using the queue manager as in any shared cluster, but with all the resources available to you.

MPI

Built on top of a Slurm cluster, this cluster also comes with an implementation of MPI (OpenMPI), ready to execute your parallel tasks.
